CARA MENJALANKAN WEBSITE SMA MUHAMMADIYAH 4 SONGGON
--------------------------------------------------
1. Pastikan Anda memiliki Python 3 dan pip.
2. Install dependensi:
   pip install flask
3. Jalankan aplikasi:
   python app.py
4. Buka di browser:
   http://localhost:5000
